# Daily Dialog dataset

> A dataset of chitchat dialogues with strong annotations for topic, emotion and utterance act.

This dataset was originally introduced in the paper: [_DailyDialog: A Manually Labelled Multi-turn Dialogue Dataset_](https://arxiv.org/abs/1710.03957). This version was download from the [ParlAI task](https://github.com/facebookresearch/ParlAI/tree/master/parlai/tasks/dailydialog).
